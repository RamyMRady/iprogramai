# iProgramAI

An AI Education Platform built with Next.js 14, TypeScript, Tailwind CSS, and shadcn/ui components.

## Features

- **30-Day AI Curriculum** - Structured learning path from AI basics to advanced prompt engineering
- **AI Playground** - Interactive chat interface to experiment with AI models
- **Prompt Library** - 100+ curated prompts for writing, coding, analysis, and creative projects
- **Authentication** - Login/signup pages with demo credentials

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- [Next.js 14](https://nextjs.org/) - React framework
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [shadcn/ui](https://ui.shadcn.com/) - UI components
- [Supabase](https://supabase.com/) - Database & auth backend
- [NextAuth.js](https://next-auth.js.org/) - Authentication
- [Lucide React](https://lucide.dev/) - Icons
